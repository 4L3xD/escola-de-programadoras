# Integração de APIs

- Telegram
- GeminiAI

# Isolamento do ambiente com uma VM
```
python -m venv vm && source vm/bin/activate
```

# Instalação de dependências
```
pip install poetry && poetry install
```

# Execução do projeto
```
poetry run python bot/telegramBot.py
```
